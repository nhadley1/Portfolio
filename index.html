Student Club Hub

A project developed by the Senior Software Engineering Class at Walsh University in the spring of 2020. The project was developed using the Vue.js framework and caused the team to adapt to develop remotely during the COVID-19 pandemic. The goal of this website is to help students find new clubs and to create a centralized place where students can view club activities.
